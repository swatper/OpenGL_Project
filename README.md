# 🎮 MuGungFlower (무궁화 꽃이 피었습니다)
> **OpenGL을 활용한 3D 액션 게임: 드라마 <오징어 게임> 기반 구현**

컴퓨터 그래픽스 수업에서 배운 OpenGL 기술들을 총동원하여 제작한 3D 게임 프로젝트입니다. 드라마 <오징어 게임>의 첫 번째 게임을 모티프로 하여 모델링부터 로직까지 직접 구현하였습니다.

## 🚀 주요 기능 (Key Features)

- **술래 시스템**
  - `Timer` 콜백 함수를 활용하여 술래가 뒤를 돌아보는 랜덤한 타이밍을 제어합니다.
  - "무궁화 꽃이 피었습니다" 10글자 카운트를 UI와 동기화하여 시각적 긴장감을 제공합니다.
- **다중 시점 (Multi-View)**
  - 숫자키(1, 2, 3) 입력을 통해 **1인칭, 2인칭, 3인칭** 시점으로 실시간 전환이 가능합니다.
- **정교한 3D 모델링**
  - OpenGL 기본 도형을 조합하여 **술래, 플레이어, 총기를 든 요원** 등의 캐릭터를 직접 모델링했습니다.
  - `glPolygonStipple`을 활용해 바닥의 잔디와 꽃 패턴을 표현했습니다.
    
- **게임 평가 및 엔딩 연출**
  - 클리어 시간에 따른 **별점 평가 시스템(Excellent/Good/Try Again)**을 포함합니다.
  - 승리 시 **폭죽 애니메이션**, 패배 시 **데드 씬(Dead Scene)** 등 동적인 연출을 구현했습니다.

## 🛠 개발 환경 및 기술 스택 (Tech Stack)

- **Language**: C++
- **Graphics Library**: OpenGL (GLUT, GLU)
- **IDE**: Visual Studio 2020
- **주요 기술**:
  - **Callback Functions**: RenderScene(Display), Timer, Keyboard, Mouse, Menu
  - **Visual Effects**: Stipple Patterns 

## 🕹 조작 방법 (Controls)

- **W / S**: 앞뒤 이동
- **마우스 왼쪽 클릭**: 술래 터치 (승리 조건)
- **숫자 1, 2, 3**: 시점 변경
- **마우스 오른쪽 클릭**: 게임 메뉴 호출 (난이도 설정 및 안내문)
